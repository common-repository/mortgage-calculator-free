= Widget Installation =

1. Access the `Widgets` page under the `Appearance` menu.
2. Drag the `Mortgage Calculator Widget` into the Widget display area of your choice.
3. Configure your settings.
4. And you're good to go!

= Shortcode Usage =

You can insert the mortgage calculator into a page or post using the short code [mortgagecalculator].
